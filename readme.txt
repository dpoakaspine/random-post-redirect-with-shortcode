Redirect user to a random post with a custom /random (or anything else) URL

== Description ==

Redirect user to a random post with a custom /random (or anything else) URL. Use the `random_post_redirect_query_var` filter to create your own random URL.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. flush your permalinks
4. go to yoursite.com/random
